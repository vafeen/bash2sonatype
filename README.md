# bash2sonatype

Bash2sonatype is a lightweight command-line utility for publishing artifacts to [Maven Central (central.sonatype.com)](https://central.sonatype.com/) without complex Gradle, Maven, or SBT configurations.

## Key Features:

### ✅ Simplicity First
- No XML/POM files - all configuration via environment variables

### ✅ Core Functionality
- Automatic POM generation from minimal configuration
- GPG artifact signing (required for Maven Central)
- Detailed logging for every step
